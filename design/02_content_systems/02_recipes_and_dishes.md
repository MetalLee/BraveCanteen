# 食谱与菜品

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义食谱解锁、菜品制作和菜品对勇者成长的作用。

## 菜品定位

菜品是本项目最核心的转化器：把食材和金币转化为勇者属性、战斗胜率和长期推进。

## 菜品类型

- 攻击菜：主要提升攻击。
- 防御菜：主要提升防御，降低每次受击伤害，适合对抗多段或持续攻击。
- 生命菜：主要提升生命，提高总容错，适合对抗重击、狂暴和机制伤害。
- 精力菜：主要提升精力，增加职业行动次数或触发额外行动。
- 元素菜：主要服务法师属性克制，命中魔物弱点时提供高额爆发。
- 复合菜：提供多属性成长，成本更高。
- 临时战斗菜：提供下一次讨伐增益，后续阶段再考虑。

## 食谱解锁来源

- 餐厅等级。
- 技能树节点。
- 魔物讨伐掉落。
- 特定战利品。

## 菜品生成规则

首版菜品生成采用“底菜 + 主食材 + 可选调味”的轻量规则。它既能让玩家感觉自己在做菜，又不会变成复杂烹饪小游戏。

| 组成 | 作用 | 示例 |
| --- | --- | --- |
| 底菜 | 决定菜品形态和基础标签 | 汤、排、饭、沙拉、冷盘、轻食盒 |
| 主食材 | 决定主要属性或职业用途 | 肉块 = 攻击，蘑菇 = 防御，小麦 = 生命，青菜 = 精力 |
| 可选调味 | 决定附加特性或元素方向 | 香草 = 复合，烈焰椒 = 火，霜盐 = 冰 |

生成逻辑示例：

| 组合 | 结果 | 说明 |
| --- | --- | --- |
| 排 + 肉块 | 烤肉排 | 最基础攻击菜。 |
| 卷 + 青菜 + 肉块 | 青菜肉卷 | 战士常用核心摄入，攻击 + 精力。 |
| 锅 / 汤 + 蘑菇 + 小麦 | 蘑菇盾锅 | 防御菜，用于多段攻击。 |
| 冷盘 + 青菜 + 霜盐 | 霜盐冷盘 | 法师冰元素菜，同时提供精力。 |
| 轻食盒 + 青菜 + 香草 | 香草轻食盒 | 弓箭手部位破坏 / 出血准备。 |
| 锅 + 魔物骨 + 蘑菇 | 骨髓护甲汤 | 盾兵防御菜。 |
| 排 + 野猪里脊 + 香草 | 野猪破甲排 | 战士破防攻击菜。 |
| 汤 + 火囊 + 小麦 | 火囊浓汤 | 法师火属性爆发菜。 |

如果玩家直接选择已解锁食谱，系统可以自动填入底菜和食材；如果玩家从食材开始组合，系统应即时预览可能生成的菜品。

## 简单有趣的烹饪体验

烹饪的趣味来自少量高质量选择：

1. 选目标：给谁吃，或为了哪个 Boss 风险做菜。
2. 选方向：攻击、防御、生命、精力、元素、出血准备。
3. 选成本：便宜稳定菜，还是稀有高收益菜。

厨房界面应提供快捷入口：

| 入口 | 功能 |
| --- | --- |
| 按职业推荐 | 显示适合战士、盾兵、法师、弓箭手、游侠的菜。 |
| 按 Boss 缺口推荐 | 显示补生命、防御、输出、元素克制、出血准备的菜。 |
| 按库存可做 | 只显示当前材料能做的菜。 |
| 一键复做 | 快速重复上一道常用菜。 |

制作前必须显示：

- 消耗食材。
- 菜品效果。
- 推荐食用对象。
- 对当前战斗预测的影响，例如“战士额外攻击 1 次”或“Boss 出血预计每次攻击自损 1%”。

## 攻击肉类菜品示例

以下菜品用于承接 [[design/02_content_systems/01_ingredients|食材]] 中的攻击型肉类，让玩家在袭击倒计时内根据魔物属性选择不同输出方案。

| 菜品 | 核心食材 | 效果倾向 | 适合场景 |
| --- | --- | --- | --- |
| 烤肉排 | 肉块 | 基础攻击 | 初始攻击菜，便宜、快速，适合首轮教学和日常补输出。 |
| 厚脂煎肉 | 厚脂兽肉 | 攻击 + 少量生命 | 面对高攻魔物时使用，牺牲部分爆发换取更稳的胜率。 |
| 筋腱肉汤 | 筋腱肉 + 小麦 | 持续攻击 | 面对高生命魔物时使用，强调长回合输出。 |
| 辣腌快烤肉 | 辣腌肉 | 临时爆发攻击 | 倒计时末段救急，适合把失败或险败推到险胜。 |
| 野猪破甲排 | 野猪里脊 + 香草 | 攻击 / 破防 | 面对高防魔物时使用，强化克制关系。 |
| 狂爪狼肉串 | 狂爪狼肉 | 高攻击 + 速度倾向 | 适合先手击杀或敏捷型勇者。 |
| 巨蜥尾肉锅 | 巨蜥尾肉 + 蘑菇 | 攻击 + 韧性 | 面对高攻、高生命目标时提供稳定输出。 |
| 龙焰战饭 | 龙焰肉 + 烈焰椒 | 极高攻击 / 下一战爆发 | 高阶稀有菜，作为阶段性目标和重反馈奖励。 |

## 魔物食材菜品示例

魔物食材菜品应强化“打怪获得新料理能力”的闭环。玩家击败一种魔物后，应获得能改变下一轮准备策略的菜品材料。

| 菜品 | 魔物食材 | 适合职业 | 效果倾向 | 来源意义 |
| --- | --- | --- | --- | --- |
| 骨髓护甲汤 | 魔物骨 + 蘑菇 | 盾兵 | 高防御 | 骨壳类魔物掉落，转化为抗压能力。 |
| 野猪破甲排 | 野猪里脊 + 香草 | 战士 | 攻击 / 破防 | 高防野猪掉落，反过来帮助对付厚甲敌人。 |
| 狼筋炖肉 | 狼筋 + 肉块 | 战士 | 持续攻击 | 狼类掉落，强化长回合输出。 |
| 火囊浓汤 | 火囊 + 小麦 | 法师 | 火属性爆发 | 火系魔物掉落，提供火克制材料。 |
| 霜腺冷盘 | 霜腺 + 青菜 | 法师 | 冰属性爆发 + 精力 | 冰系魔物掉落，提供冰克制和追加施法窗口。 |
| 硬壳磨粉沙拉 | 甲片 + 青菜 | 弓箭手 / 盾兵 | 部位破坏准备 / 防御 | 厚甲魔物掉落，帮助弓箭手理解破坏部位。 |
| 星屑侦察餐 | 星屑糖 + 香草 + 青菜 | 游侠 | 机制侦察 + 精力 | Boss 掉落，强化下一个 Boss 的可读性。 |

魔物食材菜品命名应保留来源感，但避免过度恶心。建议偏“幻想料理”而不是“猎奇料理”，例如“火囊浓汤”“骨髓护甲汤”“星屑侦察餐”。

## 精力菜品示例

精力菜用于让职业获得更多行动次数。它不应直接替代攻击菜，而是通过“多行动一次”改变战斗节奏。例如战士多砍一刀、弓箭手先破坏部位造成出血后再追射、盾兵额外挡下一次高危攻击。

| 菜品 | 核心食材 | 效果倾向 | 适合场景 |
| --- | --- | --- | --- |
| 清炒青菜 | 青菜 | 基础精力 | 初始精力菜，让玩家理解行动次数收益。 |
| 青菜肉卷 | 青菜 + 肉块 | 精力 + 攻击 | 给战士使用，适合多砍一刀形成持续输出。 |
| 脆叶盾沙拉 | 脆叶菜 + 蘑菇 | 精力 + 防御 | 给盾兵使用，适合需要多承受一次高危攻击的战斗。 |
| 香草轻食盒 | 青菜 + 香草 | 精力 + 部位破坏倾向 | 给弓箭手使用，适合先破坏部位造成出血再追射叠层。 |
| 风铃芽冷盘 | 风铃芽 + 香草 | 高精力 / 先手倾向 | 高阶精力菜，适合 Boss 战前建立关键行动窗口。 |

## 法师元素菜品示例

元素菜用于把 [[design/02_content_systems/01_ingredients|法师元素食材]] 转化为属性克制爆发。玩家应能从菜名、图标和预测提示中一眼看出它对应的元素。

| 菜品 | 核心食材 | 元素 | 效果倾向 | 适合场景 |
| --- | --- | --- | --- | --- |
| 烈焰椒汤 | 烈焰椒 + 小麦 | 火 | 火属性爆发 | 对冰、植物、再生类魔物使用。 |
| 火舌肉饭 | 烈焰椒 + 肉块 | 火 | 火属性爆发 + 攻击 | 给法师或战士混合队使用，适合快速压低回合数。 |
| 霜盐冷盘 | 霜盐 + 青菜 | 冰 | 冰属性爆发 + 精力 | 对火、狂暴、快速类魔物使用，适合让法师额外施法。 |
| 冰蘑菇浓汤 | 霜盐 + 蘑菇 | 冰 | 冰属性 + 防御 | 面对高攻火系魔物时提供克制和稳定性。 |
| 雷鸣豆炖饭 | 雷鸣豆 + 小麦 | 雷 | 雷属性穿透 | 对护盾、金属、厚甲类魔物使用。 |
| 星屑甜羹 | 星屑糖 + 月露菇 | 秘法 | 法术爆发稳定性 | Boss 战或弱点不明确时使用，提供更稳的法师输出。 |

首版建议先制作 `烈焰椒汤` 和 `霜盐冷盘`，分别对应火 / 冰克制。其他元素菜品作为后续区域、Boss 掉落或餐厅等级扩展。

## Boss 战配餐示例

Boss 战配餐应按职业职责分配，不建议让所有勇者吃同一种最高数值菜。每个职业吃什么，取决于 Boss 暴露出的风险和队伍当前缺口。

Boss 战可以把菜品分成两种用途：

| 用途 | 定义 | 示例 |
| --- | --- | --- |
| 核心摄入 | 投给本场主要输出核心，决定主要伤害来源 | 战士吃青菜肉卷，法师吃霜盐冷盘。 |
| 辅助摄入 | 投给队友，帮助核心活下来、造成持续压血或看懂机制 | 盾兵吃蘑菇盾锅，弓箭手吃香草轻食盒叠出血，游侠吃战前宴席。 |

核心摄入应优先使用稀有食材或高价值菜品；辅助摄入可以更偏稳定、便宜和针对性。

| 职业 | Boss 战主要职责 | 推荐摄入 | 解决的问题 |
| --- | --- | --- | --- |
| 盾兵 | 扛住关键伤害 | 生命菜、防御菜、脆叶盾沙拉 | 生命应对重击，防御应对多段攻击。 |
| 战士 | 持续输出 | 攻击肉菜、青菜肉卷、筋腱肉汤 | 缩短 Boss 战回合，减少进入狂暴后的时间。 |
| 法师 | 属性克制爆发 | 对应元素菜，例如烈焰椒汤、霜盐冷盘 | 命中弱点时打出高额爆发。 |
| 弓箭手 | 部位破坏 / 出血 | 香草轻食盒、精力菜、精准菜 | 破坏部位造成出血，让 Boss 每次攻击时自损。 |
| 游侠 | 机制侦察与压制 | 复合菜、战前宴席、风铃芽冷盘 | 显示 Boss 阶段风险，并降低关键机制强度。 |

## 职业最佳菜品

每个职业都应有“最佳菜品”或“招牌摄入”，让玩家形成清晰记忆：看到职业就知道它最想吃什么。最佳菜品不一定是纯数值最高，而是最能放大职业机制。

| 职业 | 首版最佳菜品 | 进阶招牌菜 | Boss 核心菜 | 设计理由 |
| --- | --- | --- | --- | --- |
| 见习勇者 | 勇者浓汤 | 香草炖肉 | 青菜肉卷 | 教学阶段先理解生命和攻击，之后用精力体验额外行动。 |
| 战士 | 青菜肉卷 | 筋腱肉汤 | 龙焰战饭 / 青菜肉卷 | 战士需要稳定输出，攻击 + 精力可以形成“多砍一刀”的核心体验。 |
| 盾兵 | 蘑菇盾锅 | 脆叶盾沙拉 | 骨髓护甲汤 / 脆叶盾沙拉 | 盾兵根据 Boss 风险在防御和生命之间切换，精力用于额外格挡。 |
| 法师 | 烈焰椒汤 | 霜盐冷盘 | 星屑甜羹 / 对应元素菜 | 法师最佳菜不是固定一道，而是命中 Boss 弱点的元素菜。 |
| 弓箭手 | 香草轻食盒 | 风铃芽冷盘 | 鹰眼香草冷盘 | 弓箭手需要精力和精准来破坏部位，追射叠加出血。 |
| 游侠 | 香草炖肉 | 战前宴席 | 星屑侦察餐 | 游侠需要复合属性和机制侦察准备，不追求单项最高输出。 |

### 招牌菜设计

| 菜品 | 推荐职业 | 食材 | 效果倾向 | 解锁 |
| --- | --- | --- | --- | --- |
| 青菜肉卷 | 战士 / 见习勇者 | 青菜 + 肉块 | 攻击 + 精力 | 初始 / 第二场战斗前 |
| 脆叶盾沙拉 | 盾兵 | 脆叶菜 + 蘑菇 | 防御 + 精力 | 餐厅等级 2 |
| 霜盐冷盘 | 法师 | 霜盐 + 青菜 | 冰属性爆发 + 精力 | 餐厅等级 2 |
| 香草轻食盒 | 弓箭手 | 青菜 + 香草 | 精力 + 部位破坏准备 | 第二场战斗后 |
| 战前宴席 | 游侠 | 肉块 + 蘑菇 + 香草 | 生命 + 防御 + 机制侦察稳定性 | 第二场战斗后 |
| 鹰眼香草冷盘 | 弓箭手 | 风铃芽 + 香草 | 高精力 + 出血叠层强化 | Boss 掉落 / 高阶食谱 |
| 星屑侦察餐 | 游侠 | 星屑糖 + 香草 + 青菜 | 机制侦察 + 精力 | Boss 掉落 |
| 星屑甜羹 | 法师 | 星屑糖 + 月露菇 | 法术爆发稳定性 | Boss 掉落 |

最佳菜品应在厨房界面以“推荐给某职业”的方式展示，例如“战士招牌”“弓箭手 Boss 菜”“游侠侦察菜”。

示例：面对“半血狂暴 + 弱冰 + 多段攻击”的 Boss：

| 职业 | 推荐摄入 | 原因 |
| --- | --- | --- |
| 盾兵 | 蘑菇盾锅或脆叶盾沙拉 | 多段攻击需要防御，精力可额外格挡。 |
| 战士 | 青菜肉卷 | 提高输出并触发额外攻击，缩短狂暴阶段。 |
| 法师 | 霜盐冷盘 | 命中弱冰，同时获得精力，可能追加施法。 |
| 弓箭手 | 香草轻食盒 | 先破坏部位造成出血，再追射叠加出血。 |
| 游侠 | 战前宴席或复合菜 | 识破狂暴阈值，降低进入半血后的风险。 |

如果本场选择战士作为核心：

- 战士优先吃青菜肉卷、筋腱肉汤或高阶攻击肉菜。
- 弓箭手吃精力或精准菜，尽早破坏部位并叠加出血。
- 盾兵吃生命或防御菜，保证核心输出时间。
- 法师只在命中元素弱点时吃元素菜，否则可以让资源给战士。
- 游侠吃复合菜，负责揭示 Boss 阶段风险。

如果本场选择法师作为核心：

- 法师优先吃命中弱点的元素菜，必要时叠加精力。
- 弓箭手负责制造出血，补足法师爆发后的持续压血。
- 盾兵优先保护法师，避免核心被重击击穿。
- 战士转为补持续输出，不抢主要稀有食材。
- 游侠负责确认弱点和 Boss 阶段机制。

## 设计准则

- 每道菜应能被玩家一眼看出属性用途。
- 高级菜品不只是数值更大，也应改变准备策略。
- 稀有食谱应强化“打魔物获得新经营能力”的闭环。
- 生命菜和防御菜必须通过预测提示区分用途，避免都被理解成“更耐打”。
- 精力菜必须显示它带来的额外行动预期，例如“战士额外攻击 1 次”或“弓箭手可先破坏部位再追射”。
- 元素菜必须显示当前是否命中魔物弱点，例如“火焰克制有效”或“未命中属性弱点”。
- Boss 战应提示某道菜是“核心摄入”还是“辅助摄入”，帮助玩家理解资源优先级。
