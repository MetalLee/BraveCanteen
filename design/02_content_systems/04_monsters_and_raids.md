# 魔物与袭击

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义魔物袭击内容、威胁等级、掉落和刷新规则。

## 袭击内容定位

魔物袭击是周期性目标和奖励来源。它应推动玩家思考当前勇者队伍是否足够强，以及下一步应该补哪类属性。

## 魔物类型方向

- 高攻魔物：考验队伍生命或防御。
- 高防魔物：考验队伍攻击。
- 高生命魔物：考验持续输出。
- 多段攻击魔物：考验队伍防御。
- 重击魔物：考验队伍生命。
- 群体魔物：后续可考验出战人数或范围能力。
- 精英魔物：提供更好掉落。
- Boss 魔物：阶段目标，带有明确机制，需要玩家针对性准备。

## 数据字段建议

| 字段 | 说明 |
| --- | --- |
| id | 魔物 ID |
| name_key | 名称文本键 |
| threat_level | 威胁等级 |
| hp | 生命 |
| attack | 攻击 |
| defense | 防御 |
| attack_count_per_round | 每回合攻击次数，可选字段 |
| heavy_strike | 是否存在重击或蓄力攻击，可选字段 |
| raid_timer_seconds | 袭击倒计时 |
| reward_table_id | 奖励表 ID |
| ingredient_drops | 魔物食材掉落列表 |
| tags | 魔物标签 |
| boss_mechanics | Boss 机制列表，仅 Boss 使用 |
| weakness_tags | 属性弱点或可破坏部位 |
| counter_role_hints | 推荐职业提示 |
| nutrition_hints | 推荐摄入提示，例如生命、防御、元素、精力 |
| core_role_hints | 推荐核心位，例如战士核心、法师核心 |

## 弱点标签

弱点标签用于支撑法师元素克制和弓箭手部位破坏。标签必须能在袭击预警或游侠侦察中展示，避免玩家靠猜。

| 标签 | 含义 | 主要应对 |
| --- | --- | --- |
| `weak_fire` | 弱火 | 法师使用烈焰椒相关菜品 |
| `weak_ice` | 弱冰 | 法师使用霜盐相关菜品 |
| `weak_lightning` | 弱雷 | 法师使用雷鸣豆相关菜品 |
| `weak_arcane` | 弱秘法 | 法师使用星屑糖相关菜品 |
| `breakable_part` | 可破坏部位 | 弓箭手破坏部位，施加出血 |
| `bleed_vulnerable` | 出血有效 | Boss 攻击时会因出血扣除百分比生命 |
| `shield_core` | 护盾核心 | 弓箭手破坏或游侠识破后处理 |

## Boss 机制方向

Boss 是阶段性考试，不应只是属性更高的普通魔物。Boss 机制必须能被预警、预测和战斗日志解释，并允许 [[design/02_content_systems/03_adventurers|游侠]] 提供机制侦察或压制。

| Boss 机制 | 玩法压力 | 推荐应对 | 游侠作用 |
| --- | --- | --- | --- |
| 狂暴阈值 | Boss 生命低于一定比例后攻击提高 | 盾兵提高承伤，战士持续输出缩短危险阶段 | 派遣前显示阈值，并降低狂暴幅度 |
| 护盾阶段 | 前几回合受到伤害降低 | 战士持续输出，弓箭手破坏护盾部位造成出血 | 显示护盾回合，缩短护盾持续时间 |
| 属性弱点 | 受到特定属性伤害提高 | 法师使用对应元素菜品 | 明确当前有效弱点，避免猜谜 |
| 蓄力重击 | 固定回合造成高伤害 | 盾兵承伤，生命菜提高容错 | 显示高危回合，并延后或降低重击 |
| 行动封锁 | 降低队伍输出或防御效率 | 游侠压制机制，复合菜提高稳定性 | 标记封锁类型，降低封锁强度 |

## Boss 配餐提示

Boss 数据应能驱动战前配餐推荐。每个 Boss 至少给出 2 到 4 个 `nutrition_hints`，帮助玩家理解小队应该摄入什么。

| Boss 特征 | nutrition_hints | 推荐说明 |
| --- | --- | --- |
| 重击 / 狂暴 | `hp_food` | 推荐盾兵或战士摄入生命菜，避免被关键伤害击穿。 |
| 多段攻击 | `defense_food` | 推荐盾兵摄入防御菜，降低每次受击伤害。 |
| 高生命 | `attack_food`、`energy_food` | 推荐战士补攻击或精力，缩短战斗回合。 |
| 属性弱点 | `element_fire`、`element_ice`、`element_lightning`、`element_arcane` | 推荐法师摄入对应元素菜。 |
| 可破坏部位 | `break_part_food`、`energy_food` | 推荐弓箭手摄入精力或精准菜，尽早破坏部位施加出血。 |
| 多段攻击且可出血 | `bleed_food`、`energy_food` | 推荐弓箭手叠出血，利用 Boss 多次攻击触发自损。 |
| 隐藏阶段机制 | `ranger_feast` | 推荐游侠摄入复合菜或战前宴席，提高侦察和压制效果。 |

## Boss 推荐核心

Boss 可以通过 `core_role_hints` 提示本场更适合谁作为主要摄入和主要输出核心。

| Boss 特征 | 推荐核心 | 理由 |
| --- | --- | --- |
| 高生命、无明显弱点 | 战士核心 | 稳定持续输出最可靠。 |
| 明确属性弱点 | 法师核心 | 元素克制能提供主要爆发。 |
| 可破坏部位、多段攻击或出血有效 | 弓箭手核心 / 战士核心 | 弓箭手用出血造成百分比持续伤害，战士承接稳定输出。 |
| 阶段机制复杂 | 游侠核心 / 战士核心 | 游侠降低机制风险，战士负责稳定收尾。 |

推荐核心不应强制玩家选择，只作为派遣预测中的建议。玩家也可以用更高数值或更好菜品打破推荐。

## 魔物食材掉落

魔物掉落应优先服务“下一轮准备”。击败某类魔物后，玩家获得的食材最好能帮助应对另一类威胁，或强化当前队伍构筑。

| 魔物 | 推荐掉落 | 对应菜品 | 设计目的 |
| --- | --- | --- | --- |
| 饥饿狼 | 狼筋、狼腿肉 | 狼筋炖肉、筋腱肉汤 | 强化战士持续输出。 |
| 石皮野猪 | 野猪里脊、硬壳肉 | 野猪破甲排 | 让高防魔物掉落破防材料。 |
| 骨壳魔物 | 魔物骨、骨髓 | 骨髓护甲汤 | 强化盾兵防御。 |
| 余烬小鬼 | 火囊、烈焰椒 | 火囊浓汤、烈焰椒汤 | 强化法师火属性爆发。 |
| 寒霜魔物 | 霜腺、霜盐 | 霜腺冷盘、霜盐冷盘 | 强化法师冰属性和精力。 |
| 厚甲精英 | 甲片、核心裂片 | 硬壳磨粉沙拉 | 支撑弓箭手部位破坏和盾兵防御。 |
| Boss | 星屑糖、龙焰肉、核心碎片 | 星屑侦察餐、龙焰战饭 | 提供阶段性高价值招牌菜材料。 |

掉落展示应说明用途，例如：“获得野猪里脊：可制作战士破防菜”。

## 设计准则

- 魔物属性应反向提示玩家需要准备什么。
- 多段攻击应提示补防御，重击或狂暴应提示补生命。
- 袭击倒计时和奖励价值要匹配威胁等级。
- 首版不做过多特性，优先验证攻防公式和奖励闭环。
- Boss 机制必须在派遣前至少提示核心风险，不允许完全隐藏到战斗后。
