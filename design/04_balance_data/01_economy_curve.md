# 经济曲线

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义金币、食材成本、菜品收益、餐厅升级和讨伐奖励之间的曲线关系。

## 基础关系

```text
金币 -> 食材 -> 菜品 -> 勇者属性 -> 讨伐奖励 -> 更高金币和稀有内容
```

## 成本原则

- 基础食材成本低，保证玩家持续操作。
- 高级食材成本更高，但单位属性收益也更高。
- 餐厅升级成本应形成阶段目标。
- 技能树成本应迫使玩家选择分支。

## 奖励原则

- 普通袭击主要给金币和普通食材。
- 进阶袭击开始给稀有食材。
- 高威胁袭击给稀有食谱和战利品概率。

## 待验证指标

- 首次餐厅升级所需时间。
- 首次讨伐成功所需准备时间。
- 食材购买后到属性提升的操作次数。
- 讨伐奖励对下一轮成长的加速比例。
