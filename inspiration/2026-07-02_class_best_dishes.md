# 职业最佳菜品灵感

日期：2026-07-02  
来源：用户与 Codex 讨论  
关联：[[design/02_content_systems/02_recipes_and_dishes|食谱与菜品]]、[[design/01_core_gameplay/04_adventurer_growth|勇者成长]]、[[design/02_content_systems/01_ingredients|食材]]

## 原始问题

用户提出：帮我设计适合上述职业最佳菜品。

## 设计方向

每个职业都应有招牌菜。最佳菜品不是单纯数值最高，而是最能放大职业机制。

| 职业 | 最佳菜品 | 机制价值 |
| --- | --- | --- |
| 见习勇者 | 勇者浓汤、香草炖肉、青菜肉卷 | 教学生命、攻击、精力三个基础概念。 |
| 战士 | 青菜肉卷、筋腱肉汤、龙焰战饭 | 攻击 + 精力，形成“多砍一刀”的持续输出核心。 |
| 盾兵 | 蘑菇盾锅、脆叶盾沙拉、骨髓护甲汤 | 防御应对多段，生命应对重击，精力触发额外格挡。 |
| 法师 | 烈焰椒汤、霜盐冷盘、星屑甜羹 | 命中属性弱点打爆发，精力提供追加施法窗口。 |
| 弓箭手 | 香草轻食盒、风铃芽冷盘、鹰眼香草冷盘 | 精力 + 精准，破坏部位并叠加出血。 |
| 游侠 | 战前宴席、星屑侦察餐、复合菜 | 提高 Boss 机制侦察和压制稳定性。 |

## 首版落地

MVP 应优先实装这些招牌菜：

- 青菜肉卷：战士招牌。
- 蘑菇盾锅：盾兵招牌。
- 烈焰椒汤 / 霜盐冷盘：法师招牌。
- 香草轻食盒：弓箭手招牌。
- 战前宴席：游侠招牌。

## 设计注意

- 厨房界面应显示“推荐职业”标签。
- Boss 战应区分核心摄入和辅助摄入。
- 稀有食材优先服务核心位招牌菜，让消耗更有重量。
