# 勇者职业职责灵感

日期：2026-06-30  
来源：用户与 Codex 讨论  
关联：[[design/02_content_systems/03_adventurers|勇者]]、[[design/01_core_gameplay/04_adventurer_growth|勇者成长]]、[[design/01_core_gameplay/06_auto_battle_system|自动战斗系统]]

## 原始想法

用户提出：

- 法师打属性克制，高额爆发。
- 战士持续输出。
- 盾兵扛伤害。
- 猎人破坏敌人弱点，限制敌人行动。后续调整为弓箭手，职责收窄为射击弱点并提高后续小队伤害。

## 整理后的设计方向

这组职业职责比单纯的攻击、防御、生命属性差异更清晰，适合作为首版勇者职业的核心框架。

| 职业 | 核心职责 | 设计价值 |
| --- | --- | --- |
| 法师 | 属性克制 / 高额爆发 | 鼓励玩家阅读魔物弱点，并准备对应元素菜品。 |
| 战士 | 持续输出 | 提供稳定伤害基线，降低战斗回合数。 |
| 盾兵 | 承受伤害 | 提高队伍容错，克制高攻魔物。 |
| 弓箭手 | 部位破坏 / 出血叠层 | 破坏敌人部位造成出血，Boss 每次攻击时因出血自损。 |

## 设计注意

- 职业差异应在派遣前预测中展示，避免玩家只能战后猜测效果。
- 法师需要魔物属性弱点系统支撑。
- 弓箭手需要魔物可破坏部位标签支撑。后续调整为“破坏部位造成出血，再次攻击造成可观伤害并叠加出血”。
- 职业职责应与菜品标签联动，例如法师偏好元素菜，战士偏好肉类持续输出菜，盾兵偏好汤锅和防御菜，弓箭手偏好破防菜和轻食。
